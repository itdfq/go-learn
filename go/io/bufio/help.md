## bufio
bufio包实现了带缓冲区的读写，是对文件读写的封装
bufio缓冲写数据

---

| 模式           |  	             含义 |
|: --------------|------------------:|
| os.O_WRONLY  |               	只写 |
| os.O_CREATE	 |              创建文件 |
| os.O_RDONLY	|                只读 |
| os.O_RDWR	|                读写 | 
| os.O_TRUNC|               	清空 |
| os.O_APPEND|               	追加 |
